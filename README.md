
# Hangouts

This app was built with React and Next.js. The goal of the app is to connect travelers with locals worldwide. Some of the technologies I used are: 

1. React Query
2. NextAuth
3. Tailwind CSS
4. Mongodb
5. Typescript
6. SSR
7. React Memo
8. React Forward Ref
9. Dynamic Routing
10. Mui
11. Google Sign-up


**Demo: https://hangout-jad2.vercel.app**
